# BarcodeImageFormat

Specifies the file format of the image.
## Allowable values

* **BarcodeImageFormat.GIF**
* BarcodeImageFormat.JPEG
* BarcodeImageFormat.PNG
* BarcodeImageFormat.TIFF
* BarcodeImageFormat.SVG
